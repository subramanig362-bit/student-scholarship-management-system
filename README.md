# Student Scholarship Management System


This project is a simple web-based system built using **HTML, CSS, and JavaScript**.


## Features
- Students can submit scholarship applications
- Applications are stored using browser localStorage
- Students can view their submitted applications


## Technologies Used
- HTML
- CSS
- JavaScript


## How to Use
1. Open index.html in a browser
2. Fill the application form
3. View your submitted entries under the "Status" section
